# Translate and Learn

Translate and Learn is a simple, browser-based communication practice game built using HTML, CSS, and JavaScript.

## Project Flow
- The project starts with **index.html**, which is the welcome page.
- The welcome page contains a **Start** button.
- When the Start button is clicked, the user is redirected to **game.html**.
- The game page contains the actual gameplay and interaction.

## How to Use
1. Open the index page in a browser.
2. Click the Start button.
3. Begin playing the game on the game page.

## Technologies Used
- HTML
- CSS
- JavaScript
